Welcome to JellyFin!

An all-in-one media hub for streaming TV Shows, Movies, and more.


Accessing the Web Portal

To access the JellyFin server, click here. You'll be directed to a login screen, where you can enter in the login information provided to you in dms from a Jellyfin Manager. You only get 3 attempts at signing in before it disables your account. If this happens ping a Jellyfin Manager.

Note that this login information is your eyes only. Sharing this information will result a removal from the Jellyfin server, and possibly removal of access from the entire server!

After you login, you should see three things: Movies, Shows, and Books. Select one, then it'll open a whole array of movies, shows, or books (depending on what you selected). To access the media you want to view, press on the play button.

Do keep in mind that you can only have 2 sessions active per user. This means if you try to connect to the server on 3 devices, the third device will not connect to the server, and will return an error.

If your watching gets interrupted, your place will be saved as long and you're 5 percent into a movie or episode. Books are a different story, and Jellyfin will save your spot as you flip through pages.


Discord Server

You can join the Drap's Labs Discord server here.

The Discord server is optional. However, this server will update you on the status of the server, like downtimes, new media, and ect.

Opon joining, if you do not get assigned the Jellyfin access role, then please ping the same Jellyfin Manager who provided your password.


Changing Your Password

It's VERY recommended to change your password after you login for the first time. The provided password from a Jellyfin Manager is supposed to be a temporary password, and is not secure.

To change your password, follow these three steps: 3 lines in the top left -> settings -> profile -> enter current password -> enter new password + confirmation.

Please note that if you forgot your password, you can dm a Jellyfin Manager and they'll reset it for you.


Help/Assistance

If you encounter a Jellyfin problem, like a fatal playback error, please ping a Jellyfin Manager in the Discord server.

If you encounter a Cloudflare error, ping a Server Manager.

Enjoy!
